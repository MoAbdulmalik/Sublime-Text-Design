Navigate source using Goto Definition Here the mouse is hovered over a symbol to jump to the definition. The keyboard shortcut F12 can be used when the caret is located on a symbol,
or Goto Symbol in Project can be opened via Ctrl+Shift+R to jump to the definition of any symbol in the project.

Some features our users love:

GOTO ANYTHING

Use Goto Anything to open files with only a few keystrokes, and instantly jump to symbols, lines or words. Triggered with Ctrl+P, it is possible to: Type part of a file name to open it. Type @ to jump to symbols, # to search within the file, and : to go to a line number. These shortcuts can be combined, so tp@rf may take you to a function read_file within a file text_parser.py. Similarly, tp:100 would take you to line 100 of the same file.
